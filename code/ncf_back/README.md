# ncf_back

Spring Boot backend for the NCF recommendation demo.

## APIs

- `GET /api/items`
- `GET /api/items/{itemId}`
- `GET /api/recommendations/users/{userId}`
- `GET /api/media/items/{itemId}/preview`
- `GET /api/media/items/previews?itemIds=1,2,3`
- `GET /api/favorites/users/{userId}`

## Personalized Recommendations

- Recommendations are now generated by a unified online `NCF+Attention` scoring pipeline.
- The backend no longer falls back to a popularity list when filling recommendation slots.
- Every active user is refreshed to exactly 10 recommendation rows in `rec_results` at startup.
- Returned `score` values are normalized to `0-1` and can be compared directly inside the same response.
- Missing `user_embeddings` and `item_embeddings` are backfilled into the database under model version `ncf_attention_v2_20260314`.

## Real Music Catalog

- The backend now stores a real music catalog for all current `items` rows.
- `items.title`, `items.artist_name`, `items.album_name` and `item_media.preview_url`
  are synced from `https://jkapi.com/api/music`.
- `item_media.preview_url` stores the API `music_url`, so the frontend preview button
  can play the returned audio directly.
- The preview metadata API now returns a local playable URL: `/api/media/items/{itemId}/play`.
  Use that URL for the audio element `src`; the backend will refresh and redirect to the latest `music_url`.
- When a preview URL becomes stale, the backend refreshes it again from `jkapi`
  using the stored query key in `external_item_no`.

## Config

Main config is in [application.yml](D:/NCFResearch/code/ncf_back/src/main/resources/application.yml).

- `app.music.jkapi`
  - API base URL, key, platform and preview refresh window.
- `app.music.catalog.auto-sync-on-startup`
  - If the database still contains placeholder songs, startup will sync them to real songs.
- `app.recommendation`
  - Controls startup refresh, top-k size, scene name, and the online NCF+Attention model version.

## Run

Preferred commands:

```bash
mvn spring-boot:run
```

or:

```bash
mvn -DskipTests package
java -jar target/ncf-back-1.0.0.jar
```

If IntelliJ launches the main class without Spring Boot dependencies on the classpath,
use the shared Maven run configuration `NcfBack Maven`.

## Manual Sync

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\sync_real_music_catalog.ps1
```
